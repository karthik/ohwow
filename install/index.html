Step by step

Three lines, start to together.

It lives inside the tools you already use. After setup, the whole thing is a single word.

1
ADD

Add the marketplace

Point Claude Code at the ohwow plugin marketplace. One time only.

/plugin marketplace add cgreene/wonder
2
INSTALL

Install /wonder

Install the plugin from that marketplace. Now /wonder is in every session.

/plugin install wonder@ohwow
3
WONDER

Run it when it gets quiet

It summarizes the session, scrubs anything sensitive, and — once you approve — drops you in with people on the same thing.

/wonder
→ room: protein-folding · cryo-em
# you are not working alone

Nothing leaves your machine until you say so.

/wonder summarizes and scrubs locally, shows you the exact keywords, and only sends them after you approve. ORCID identity is optional and coming soon — today it matches on keywords.
